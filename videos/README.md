1. Download or create MP4 video files for each movie/show.
2. Rename them exactly as listed above.
3. Place them in this `videos/` folder.
4. The videos should be in MP4 format for compatibility with the HTML5 video player.

## Note:
If you don't have the actual videos, you can use placeholder videos or trailers from YouTube (but you'll need to download them as MP4 files).
